### RBAC Implementation

#### Data Filtering Logic:
```javascript
// Redux Selector with RBAC
const selectFilteredTrainings = (state) => {
  let filtered = state.trainings.data;
  
  // Filter by allowed states
  if (state.auth.allowedStates.length > 0) {
    filtered = filtered.filter(t => 
      state.auth.allowedStates.includes(t.state)
    );
  }
  
  // Filter by allowed partners
  if (state.auth.allowedPartners.length > 0) {
    filtered = filtered.filter(t => 
      state.auth.allowedPartners.includes(t.partnerId)
    );
  }
  
  return filtered;
};
```

#### Feature Access Control:
- **Create Training**: NDMA Admin only
- **Edit Training**: NDMA Admin + Partners (own programs)
- **Delete Training**: NDMA Admin only
- **Generate Reports**: All authenticated users
- **View AI Insights**: All authenticated users
- **Bulk Actions**: NDMA Admin only
